# AI 辅助开发日志

## 1. 任务目标
开发一个基于 Python 的简易文档自动排版工具，能够清理文本中的冗余格式。

## 2. AI 工具选择
- **工具名称**：通义千问 (Qwen)
- **选择理由**：擅长代码生成与逻辑梳理，能快速提供可运行的 Python 脚本。

## 3. 交互过程记录
- **Prompt (提示词)**：
  > "请帮我用 Python 写一个文档自动排版工具，要求能读取 txt 文件，清理多余空格和空行，并提供一个简单的数字菜单。"

- **AI 生成结果**：
  AI 提供了一个包含 `clean_text` 函数和 `while True` 菜单循环的完整脚本。使用了 `re` 正则表达式模块来处理字符串替换。

## 4. 修改与优化
- **问题发现**：原始代码在处理中文路径时可能会报错。
- **优化操作**：在 `open()` 函数中显式添加了 `encoding='utf-8'` 参数，确保跨平台兼容性。
- **功能确认**：测试了连续空格替换逻辑，确认符合任务书要求。
