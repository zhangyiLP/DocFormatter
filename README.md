# 📄 DocFormatter - 智能文档格式化工具

> 一个基于 Python 的轻量级文本处理工具，专注于自动化文档清洗与排版优化。

## 🚀 项目简介
在日常学习和工作中，我们经常遇到从网页复制的文本格式混乱、缩进不统一、多余空行满天飞的情况。**DocFormatter** 就是为了解决这些痛点而生的。

本项目采用 **AI 辅助编程 (Human-in-the-loop)** 模式开发，通过 Python 脚本实现了对 `.txt` / `.md` 文件的自动化处理。它不仅能“读懂”你的文件，还能像专业的排版编辑一样，一键完成清洗和美化。

### ✨ 核心功能
- **🧹 冗余清理**：自动识别并删除文档中连续的空行、行首尾的多余空格以及不可见的特殊字符。
- **📏 智能排版**：一键统一段落缩进（支持 Tab 转空格），规范标点符号的全角/半角格式。
- **🛡️ 安全读写**：内置编码检测机制，有效防止文件读取时的乱码问题，确保原文件数据安全。

## 🛠️ 技术栈
- **语言**: Python 3.x
- **核心模块**: `os` (文件路径管理), `re` (正则表达式清洗), `sys` (系统交互)
- **开发模式**: AI 辅助编程 (Prompt Engineering + Code Review)
